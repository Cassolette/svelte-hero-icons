# svelte-hero-icons

## Install

NPM
```bash
npm install --save-dev svelte-hero-icons
```

## Usage

```html
    <script>
        // Only import what you need!
        import { MdArrowUp, SmFilter, ... } from 'svelte-hero-icons'
    </script>
    
    <MdArrowUp size="24" />
    <SmFilter size="1.5x" />
```

See all icons here: https://github.com/refactoringui/heroicons
